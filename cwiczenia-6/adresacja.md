Struktura adresu IP
-------------------

```192.168.100.192```
```255.255.255.0```

Adres sieci
-----------

1. 192.168.100.192/24
2. 192.168.0.0/24
3. 192.168.100.0/24

Adres rozgłoszeniowy
-----------

1. 192.168.100.255
2. 192.168.0.255
3. 192.168.100.255


Podział na równą ilość podsieci
-------------------------------

```2^S >= n```

Chcemy 7 -> 2^3 >= 7

Maska -> 255.255.255.11100000 -> 255.255.255.224 -> /27

Parametry na podstawie adresu
-----------------------------

Mając dany adres hosta i maskę znajdź:
  * adres sieci
  * początkowy adres hosta w sieci
  * liczbę hostów w zadanej podsieci ```2^(32 bity - bity maski maska) - 2 (siec i broadcast)```
  * końcowy zakres adresu hosta w sieci
  * adres rozgłoszeniowy
  
  ------------------------------
| Parametr |  wartość   | 
| --------- |:-------------| 
| ``ip``    | 192.168.1.145| 
| ``maska``   | 255.255.255.128 | 
| ``adres sieci``   | 192.168.0.0/17 |
| ``liczba hostów``   | 32766 |
| ``host - min``   | 192.168.0.1 | 
| ``host - max``   | 192.168.0.254 | 
| ``broadcast``   | 192.168.127.255 |
